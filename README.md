# Gotham City Police Department
Idealizado no curso de T.I (SENAC TECH), esse projeto tem como objetivo a criação de um site para o Departamento de Polícia de Gotham City, pertencente ao universo fictício DC. Isso não é muito diferente do que pode ser feito para um cliente real, seja departamento, escritório, ou qualquer outro lugar que lida com gerenciamento de dados. Utilizando HTML e CSS, visamos aplicar um estilo único às páginas que se aproximam da estética dos quadrinhos. E não menos importante, nos esforçamos para  criar um layout responsivo, que seja adptável a todos os tipos de tamanhos de tela.
